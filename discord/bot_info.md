# Configuração do Bot Discord - One Habbo

Este diretório contém informações e scripts relacionados ao Bot do Discord da comunidade.

## Informações do Bot

- **Nome:** One Habbo Bot (Exemplo)
- **Prefixo:** `!` (ou `/` para slash commands)
- **Linguagem:** Node.js (Discord.js) / Python (Discord.py)

## Credenciais (NUNCA COMITAR O TOKEN REAL)

Crie um arquivo `.env` nesta pasta para armazenar as chaves secretas.

```env
DISCORD_TOKEN=seu_token_aqui
CLIENT_ID=seu_client_id_aqui
GUILD_ID=id_do_servidor_principal
```

## Funcionalidades Planejadas

1.  Verificação de conta Habbo.
2.  Notificações de notícias do site.
3.  Moderação automática.
4.  Integração com rádio.

---

**Nota:** Mantenha o token seguro e nunca o compartilhe publicamente.
